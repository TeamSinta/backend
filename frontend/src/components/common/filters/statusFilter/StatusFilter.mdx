## StatusDropdownFilter Primary

- When the status is not selected

### How to use

```jsx

const args = {
  status: null
}

<StatusFilter {...arg} />
```

<Canvas>
  <Story of={StatusFilterStories.PrimaryStatusFilter} />
</Canvas>

## DropdownFilter Active

- When the status is `Active`.

### How to use

```jsx

const arg = {
  status: StatusDropdownFilter.ACTIVE,
}

<StatusFilter {...arg} />
```
