# IconBtn

---

### A button with only an icon.

- To use the `<ElWrap w={56} h={134}></ElWrap>` element, simply input the appropriate numbers for the 'w' and 'h' parameters to adjust the size accordingly.
- To use the enum type `BackgroundColor.WHITE` or `BackgroundColor.ACCENT_PURPLE` to assign a class name and change the color accordingly.

## IconBtnL

### For icon button L, there are three different types available:

- Vertical elongated type
- Horizontal elongated type
- Square type

### How to use

```jsx

const arg = {
  icon: <PlusIcon />,
  disable: true,
  className: BackgroundColor.WHITE or className: BackgroundColor.ACCENT_PURPLE,
  onClick: action("Click Handler"),
}

//Vertical elongated
<ElWrap w={56} h={134}>
  <IconBtnL {...args} />
</ElWrap>

//Horizontal elongated
<ElWrap w={93} h={40}>
  <IconBtnL {...args} />
</ElWrap>

//Square type
<ElWrap w={40} h={40}>
  <IconBtnL {...args} />
</ElWrap>
```
