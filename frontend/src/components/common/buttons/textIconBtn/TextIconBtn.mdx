## TextIconBtnXL

### For Text Icon button XL:

### How to use

```jsx

const arg = {
  label: "Create Custom Role",
  icon: <AsteriskIcon />,
  disable: true or false,
  className: BackgroundColor.WHITE or BackgroundColor.ACCENT_PURPLE,
}

<TextIconBtnXL {...args} />

```
