## TextBtnL

### For Text button L, there are two different types available:

- width 223px type
- width 119px type

### How to use

```jsx

const arg = {
  label: "Create",
  disable: true or false,
  className: BackgroundColor.WHITE or BackgroundColor.ACCENT_PURPLE,
}

<ElWrap w={223}>
  <TextBtnL {...args} />
</ElWrap>

<ElWrap w={119}>
  <TextBtnL {...args} />
</ElWrap>

```
